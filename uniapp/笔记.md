一个页面多组件：  
第一步：  
```
<view>
    <view>
        <view :class="{active:active==index}" //选定后的类
        @click="onclick(index)"//点击事件
        v-for="（item,index）in data" :key="index"//遍历循环显示内容
        >
            {{item}}
        </view>
    </view>
</view>
```
第二步：  
```
data(){
    return{
        active=0,
        data:["h","j"]
    }
}
```
第三步：  
```
mothods:{
    onclick(id){
        this.active=id//判断哪个页面
    }
}
```
最后通过v-if方法判断跳转哪个页面
