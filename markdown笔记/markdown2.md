4表格
===
    表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行  
    
例子： 

    |  表头   | 表头  |  
    |  ----  | ----  |  
    | 单元格  | 单元格 |  
    | 单元格  | 单元格 | 

---

|  表头   | 表头  |
|  ----  | ----  |
| 单元格  | 单元格 |
| 单元格  | 单元格 |

    设置对齐方式：
    1. -: 设置内容和标题栏居右对齐
    2. :- 设置内容和标题栏居左对齐
    3. :-: 设置内容和标题栏居中对齐
例子：  
    | 左对齐 | 右对齐 | 居中对齐 |  
    | :-----| ----: | :----: |  
    | 单元格 | 单元格 | 单元格 |  
    |单元格 | 单元格 | 单元格 |  

---

| 左对齐 | 右对齐 | 居中对齐 |
| :---------| -------: | :--------: |
| 单元格单元格单元格 | 单元格单元格单元格 | 单元格单元格单元格 |
| 单元格 | 单元格 | 单元格 |
